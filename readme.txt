== Description ==

This plugin is developed to perform a visual calculation of the fees to pay for a product and service. The months and their respective interest for the calculation can be entered in the Backed.

When activating the plugin, an item will appear in the administrator menu that says "Calculate Quotas", we click and it redirects us to a simple form where we will add the values that will be used for the calculation.

On the product page, a select will appear with the values of the months added in the administrator, we choose a month and the calculation is automatically performed
